# Healthminder
## Overview
Healthminder is a simple, browser-based health reminder web application that
helps users keep track of medications, appointments, exercises, and other health
routines[1]. Built with plain HTML, CSS, and JavaScript, it provides a clean
interface for creating, viewing, editing, and deleting reminders. All data is stored in
your browser's local storage, ensuring privacy and persistence without requiring a
backend server.
## Features
- Add new reminders for medications, appointments, exercises, and other custom
types
- View an upcoming reminders list with scheduled dates and details
- Edit, delete, and mark reminders as completed or incomplete
- Responsive and accessible design for use on both desktop and mobile devices
- No signup or authentication required; runs completely client-side for fast and
secure use
## How to Run
1. Download or clone the repository containing `index.html`
2. Open the `index.html` file in any modern web browser (Chrome, Firefox, Edge,
Safari)
3. The app will load instantly and store reminders using your browser's local
storage
4. No setup or installation required; you can start adding and managing reminders
immediately
## Usage Instructions
- **Add Reminder:** Use the form on the left of the main interface. Fill out the
name, details (such as dosage or location), date, time, and type. Click "Add
Reminder" to save
- **Edit/Complete/Delete:** Each reminder in the list shows action buttons. You
can edit the entry, mark as done/not done, or delete using these controls
- **Filtering/Sorting:** Reminders are automatically sorted by completion status
and scheduled date
## Technologies Used
- HTML (semantically structured for accessibility)
- CSS (custom gradients, media queries, card-based layout)
- JavaScript (local storage operations, event handling, dynamic rendering)
- [Remixicon](https://remixicon.com/) for UI icons
- [Google Fonts: Inter](https://fonts.google.com/specimen/Inter) for typography
## Accessibility & Responsiveness
The interface is designed to be accessible, with keyboard and screen-reader
compatibility. Responsive layout ensures usability on both mobile and desktop
screens
